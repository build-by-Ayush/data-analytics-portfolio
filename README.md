## 📂 Featured Projects

Instead of just showing standard charts, my portfolio focuses on the "why" and "how" behind each project, including data modeling and ETL pipelines. Here is what you will find inside:

1. **E-Commerce Funnel Analytics:** A diagnostic tool built with Python and Streamlit to track customer drop-offs and financial leakage at checkout.
2. **BMTC Transit Service Gap Analysis:** A geospatial Power BI project that maps out population density against bus stop locations to find underserved areas in Bengaluru.
3. **Intelligent Anomaly Monitoring:** A time-series monitoring platform that detects unusual spikes in continuous data without relying on rigid manual thresholds.
4. **Global Sales & Demographics:** A robust Power BI Star Schema model to track global revenue, return rates, and test pricing scenarios.
5. **T20 Cricket Performance Analytics:** A predictive framework to select an optimized "Final 11" team based on role-specific metrics rather than just basic averages.
6. **CareSync Adaptive Notifications:** An on-device database and ML project analyzing why users dismiss notifications and optimizing when to send them.
